# Publishing Checklist #

If updating projects you must
1. Add project description to projects.md
2. `git add projects.md`
3. `git commit`
4. `git push`
5. execute `bash bob.sh` in root

If writing new essay you must
1. Write in markdown format in `essays` dir
2. Add essay description in writing.md
3. `git add writing.md`
4. `git add {essay_file}`
5. `git commit`
6. `git push`
7. execute `bash bob.sh` in root
